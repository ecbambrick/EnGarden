# En Garden
A simple platformer for the AGDG Spring Fling game jam

***

### Instructions

 * Jump on enemies to kill them
 * Plant seeds dropped by enemies to grow plants
 * UP: Jump
 * DOWN: Plant/Uproot seeds
 * LEFT/RIGHT: move
